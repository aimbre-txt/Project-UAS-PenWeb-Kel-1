# HomeElectro - Toko Alat Elektronik Rumah

## Latar Belakang
HomeElectro adalah platform penjualan alat elektronik rumah tangga yang dirancang untuk memudahkan pengguna dalam mencari, membandingkan, dan membeli produk elektronik secara online. Website ini menyediakan berbagai kategori produk seperti televisi, kulkas, mesin cuci, rice cooker, serta aksesoris elektronik pendukung lainnya. Pengembangan website ini merupakan bagian dari proyek pembelajaran di bidang pemrograman dan pengembangan web, dengan tujuan meningkatkan efisiensi penjualan produk elektronik serta memberikan pengalaman berbelanja yang aman, nyaman, dan informatif bagi pengguna.

## Deskripsi Fitur Utama
Website HomeElectro dilengkapi dengan berbagai fitur untuk memudahkan pengguna dan pengelola:

### Fitur untuk Pengguna:
- **Katalog Produk Lengkap**: Menampilkan berbagai produk elektronik rumah tangga dengan gambar dan harga.
- **Detail Produk**: Halaman detail produk dengan deskripsi lengkap, gambar, dan tombol beli.
- **Keranjang Belanja**: Fitur keranjang untuk menyimpan produk yang ingin dibeli.
- **Pencarian dan Filter**: Memungkinkan pengguna mencari produk berdasarkan kategori atau harga.
- **Halaman Tambahan**: Login, chat live, riwayat pembelian, garansi, kontak, dan tentang kami.
- **Tampilan Responsif**: Website dapat diakses dengan baik di berbagai perangkat (desktop, tablet, mobile).

### Fitur untuk Admin:
- **Manajemen Produk**: Menambah, mengedit, dan menghapus data produk.
- **Pengaturan Stok**: Mengelola stok barang.
- **Proses Pesanan**: Memproses pesanan pelanggan.
- **Laporan Penjualan**: Memantau laporan penjualan sebagai bahan evaluasi bisnis.

## Teknologi yang Digunakan
- **HTML**: Struktur dasar halaman web.
- **CSS**: Styling dan layout website (termasuk file style.css).
- **JavaScript**: Interaktivitas dan fungsi dinamis (termasuk file main.js).
- **Bootstrap 5.3.2**: Framework CSS untuk komponen UI dan responsivitas.
- **Bootstrap Icons**: Ikon untuk elemen UI.
- **Box Icons**: Ikon tambahan untuk navigasi dan elemen lainnya.
- **Swiper JS**: Library untuk slider gambar di halaman utama.

## Cara Menjalankan
1. Pastikan semua file proyek (HTML, CSS, JS, dan folder img) berada dalam satu direktori.
2. Buka file `index.html` menggunakan browser web (seperti Chrome, Firefox, atau Edge).
3. Website akan berjalan secara lokal tanpa perlu server tambahan, karena ini adalah proyek frontend sederhana.
